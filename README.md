# AI Resume Analyzer API

AI Resume Analyzer is a backend API built with FastAPI that analyzes resumes and provides feedback on skills, experience, and job match.

## Features

- Upload resume text
- AI analyzes resume
- Suggests improvements
- Swagger API documentation
- FastAPI backend

## Technologies

- Python
- FastAPI
- Uvicorn
- OpenAI API
- Pydantic
